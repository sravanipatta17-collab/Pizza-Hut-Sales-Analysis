# Pizza-Hut-Sales-Analysis
This project analyzes Pizza Hut sales data to uncover key business insights using SQL for data querying and transformation, and Tableau for interactive visualization. The analysis focuses on sales trends, top-performing pizzas, revenue contribution, customer ordering patterns, and time-based performance.
